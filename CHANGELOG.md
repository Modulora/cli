## modulora@0.1.0

### Initial release: the verified install engine

`search`, `info`, `add` (content-digest verification — refuses mismatches),
`verify`/`diff`/`update` over `modulora.lock` (never discards local edits),
device-flow `login`/`whoami`/`logout`, entitlement-gated paid installs,
install receipts, `publish` from a local manifest, and a read-only MCP
server (`modulora mcp`) for AI agents.
